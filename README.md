# Super Store Sales Dashboard

## 📊 Project Overview
This is an interactive Excel dashboard that provides comprehensive sales analytics for a super store chain across multiple regions and time periods. The dashboard enables data-driven decision-making through real-time visualization of key business metrics.

## ✨ Features

### Key Performance Indicators (KPIs)
- **Total Sales**: $58,20,182
- **Sales Average**: $582.37 per transaction
- **Total Transactions**: 9,994
- **Total Quantity**: 25,123 units sold

### Visualizations & Analytics
- **Ship Mode Sales %**: Breakdown of sales by shipping method (First Class, Second Class, Same Day, Standard Class)
- **Region-wise Transactions**: Donut chart showing transaction distribution across Central, East, South, and West regions
- **Monthly Quantity Ordered**: Trend line tracking quantity patterns throughout the year (Jan-Dec)
- **Top 5 Cities Sales**: Bar chart comparing sales performance across New York, Los Angeles, Philadelphia, Seattle, and San Francisco
- **Category-wise Transactions**: Performance analysis for Technology, Office Supplies, and Furniture categories
- **Consumer Type Quantity Ordered**: Segmentation by Home Office, Corporate, and Consumer buyer types

### Interactive Filters
- **Regional Filters**: Central, East, South, West
- **Year Filters**: 2014, 2015, 2016, 2017
- Dynamic dashboard that updates based on selected filters

## 🛠️ Skills Demonstrated
- Advanced Excel formulas (SUMIF, VLOOKUP, INDEX-MATCH)
- Pivot Tables & Pivot Charts
- Data Visualization & Chart Design
- Interactive Dashboards with Slicers/Filters
- Data Analysis & Business Intelligence
- Dashboard Layout & UI/UX Design

## 📈 How to Use
1. Download the Excel file
2. Use the filter buttons on the left sidebar to select:
   - **Regions**: Click on any region (Central, East, South, West)
   - **Years**: Select year(s) from 2014-2017
3. All charts and metrics update automatically based on your selection
4. Hover over charts to see detailed values

## 💡 Key Insights
- **Standard Class** is the most popular shipping mode (58% of sales)
- **Consumer segment** drives the highest quantity ordered (13,030 units)
- **Central region** leads in transactions with 2,323 total transactions
- **Monthly trends** show seasonal variations, with peaks in September and October
